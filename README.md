WeChatMsgArchiveApp 是一个 Windows 桌面工具（注意：不是微信小程序，虽然放在"微信小程序"目录下，但它是一个 Python 桌面应用），用于解密并导出电脑版微信的本地聊天记录。注意事项
仅限 Windows — 解密功能通过读取微信进程内存获取密钥，依赖 Windows API
仅限本人数据 — 项目声明只应处理自己或已获授权的微信数据
导出完整性依赖本地缓存 — 只能导出电脑微信已同步到本机的聊天记录，手机上的记录需要先在电脑微信同步
附件不完整 — 图片、语音、文件需要本地有缓存才能导出
项目整体是一个完整的、生产级的微信聊天记录导出工具，覆盖了解密→解析→多格式导出的全链路，还额外支持 AI 训练数据生成
